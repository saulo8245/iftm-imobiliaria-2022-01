Orientações quanto ao uso do Sistema.

Esse sistema imobiliário é um sistema para encontrar imovéis para aluguel ou compra, ou se registrar como um dono de imóvel e registrar seus imovéis para aluguel ou venda.

A seguir listaremos as páginas para o registro de:

  • Imóveis para aluguel ou venda.
  • Clientes no sistema.
  • Imagens dos imóveis para aluguel ou venda.
